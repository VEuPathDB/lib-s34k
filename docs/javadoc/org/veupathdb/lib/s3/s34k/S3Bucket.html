        Downloads the remote object at path into the specified local file.

        If the target file does not exist, it will be created.

        If the target file already exists, it will be truncated before downloading the object from the S3 bucket.

        Parameters:
        path - Path to the target object in the root bucket.
        localFile - Reference to the local file into which the contents of the remote object will be copied.

      • putDirectory

         abstract S3Object putDirectory(String path)

        Creates an empty directory at the given path.

        Warning: Not all S3 implementations allow the creation of or hold onto empty directories.

        Note: Some S3 implementations allow the creation of directories simply by attempting to put an object with a directory path. For instance, putting an object at "foo/bar/fizz" may create the directory structure "foo/bar" automatically.

        Parameters:
        path - Path to the object to create.
